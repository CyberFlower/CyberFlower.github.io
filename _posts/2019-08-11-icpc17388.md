입출력 연습문제입니다. 같이 숭고한 운영진을 했던 evenharder 님의 python 코드에 감명을 받아 evenharder 님의 코드를 소개합니다.

```python
a=[int(x) for x in input().strip().split()]
print("OK" if sum(a)>=100 else "Soongsil Korea Hanyang".split()[a.index(min(a))])
```
